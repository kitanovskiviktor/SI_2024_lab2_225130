### Цикломатска комплексност

Цикломатската комплексност на овој код е 10, каде што ја добив преку следната формула
(Број на ребра - Број на јазли) + 2, во случајов имаме 31 ребро, 23 јазли.
(31 - 23) + 2 = 8 + 2 = 10
